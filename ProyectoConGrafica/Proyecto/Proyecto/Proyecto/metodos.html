Método de Bisección


¿Qué es el Método de Bisección?


El método de Bisección está basado en un concepto muy sencillo, que consiste en tomar un intervalo que encierre la raíz que deseamos calcular, luego subdivimos dicho intervalo por la mitad y tomamos el sub-intervalo que contiene la raíz, descartando la otra mitad que no la contiene. Repitiendo este proceso podemos obtener el valor de la raíz con la precisión que deseemos. Para ello, partimos de un intervalo [a,b] arbitrario, que contenga una única raíz real de la función. Es de hacer notar que este método no permite hallar raíces dobles, debido a que este tipo de raíces toca de manera tangencial el eje de las x, por otra parte este método, no es capaz de diferenciar entre una raíz y una singularidad.





Explicación paso a paso:



  • Paso 1: Elija las conjeturas iniciales a, b y la tasa de tolerancia e.
  • Paso 2: Si f(a)f(b) >=0, entonces la raíz no se encuentra en este intervalo. Por tanto, no habrá solución.
  • Paso 3: Encuentra el punto medio, c = (a+b)/2.
    • Si el valor de la función del punto medio f(c) = 0, entonces c es la raíz. Vaya al paso 5.
    • Si f(a)f(c) < 0, la raíz se encuentra entre a y c. Luego establezca a = a, b = c.
    • De lo contrario, establezca a = c, b = b.
  • Paso 4:Si el error absoluto es mayor que la tasa de tolerancia o (b-a) > e, vaya al paso 3.
  • Paso 5:Muestre c como la raíz aproximada.

Ejemplo del método de bisección



  • Tenemos que encontrar la raíz de la siguiente función: ( f(x) = x^3 - x^2 + 2)
  • Paso 1: Asumamos
    • a = -10,
    • b = 10, y
    • e = 1% o 0.01
  • Paso 2: Ahora comprobaremos si f(a)f(b)>=0.
    • f(a) = f(-10) = (-10)^3 – (-10)^2 + 2 = -1098
    • f(b) = f(10) = (10)^3 – (10)^2 + 2 = 902
    • f(a)f(b) = f(-10)f(10) = (-1098)(902) < 0
    Por tanto, la raíz de la función anterior se encuentra en este intervalo ([-10, 10]).
  • Paso 3: Entonces se calculará primero el punto medio c.


    • (1.) si f(c) = 0
      • f(c) = f(0) = (0)^3 – (0)^2 + 2 = 2 ≠ 0
    • (2. ) si ( f(a)f(c) < 0 ):
      • f(c)f(a) = 2*(-1098) < 0
    La condición se cumple. Para la próxima iteración, los valores serán,
    • una = -10
    • segundo = c = 0
  • Paso 4: Como ( (b-a) = (0-(-10)) = 10 > 0.05), se repetirá el proceso. Las siguientes iteraciones se muestran en la tabla.


  • Paso 5: En la undécima iteración, la condición del paso 11 será falsa. Por tanto, la raíz de esta ecuación es -4
